# 💳 Credit Card Fraud Detection System

An advanced Machine Learning-based system to detect fraudulent credit card transactions with a real-time Streamlit interface.

---

## 🚀 Features
- Detects fraudulent transactions using ML models
- Real-time prediction using Streamlit UI
- Handles imbalanced datasets
- Data preprocessing and scaling
- User-friendly interface

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

## 📊 Model Used
- Random Forest Classifier

---

## 📁 Project Structure
- `app.py` – Streamlit app  
- `train_advanced_model.py` – Model training  
- `test_sample_predict.py` – Testing script  
- `random_forest_model.pkl` – Saved model  
- `scaler.pkl` – Data scaler  
- `requirements.txt` – Dependencies
